3.0 Übungen zu Funktionen, Strukturen und Unions

  1. Aufgabe

    Schreiben Sie eine Funktion, die den Absolutwert (Wert ohne Vorzeichen) eines double-Wertes zurückliefert.

  2. Aufgabe

    Schreiben Sie eine Funktion "vorzeichen", die einen Integer-Wert zurückliefert und einen double-Parameter übernimmt. Ist der double-Wert größer Null, soll 1, ist er kleiner Null, soll -1 und ist er gleich Null, so soll Null zurückgegeben werden.

  3. Aufgabe

    Schreiben Sie eine Funktion "hoch", die einen double-Wert potenziert und das Ergebnis zurückliefert. Der Prototyp sieht folgendermaßen aus: 

      double hoch(double wert, int potenz);
    Bei einer Potenz = 0 soll der Wert 1, bei einer negativen Potenz der Fehlerwert -1 zurückgegeben werden.

  4. Aufgabe

    Schreiben Sie eine Funktion, die mehrere Integer-Werte von der Tastatur entgegennimmt und den größten zurückliefert.

  5. Aufgabe

    Schreiben Sie ein Programm, das zur Berechnung des Durchschnittswertes eine Funktion verwendet. Es ist hierbei unerheblich, ob die Zahlen vom Benutzer eingegeben oder der Funktion als Feld übergeben werden.

  6. Aufgabe

    Schreiben Sie eine Funktion, die in einer Zeichenkette ein beliebiges Zeichen sucht und es überall in dieser Zeichenkette durch ein anderes Zeichen ersetzt. Beispiel: Alle e in einem Satz durch x ersetzen.
    Benutzen Sie den folgenden Prototyp: 

    void ersetze_zeichen(char *s, char alt, char neu);

  7. Aufgabe

    Erweitern Sie die Funktion aus der vorherigen Aufgabe derart, daß Sie angeben können ab dem wievielten Vorkommen des Zeichens, das gesucht werden soll, ersetzt wird und wieviele darauffolgende Vorkommen ersetzt werden. Beispiel: Alle e ab dem zweiten Vorkommen in einem Satz durch x ersetzen, maximal jedoch vier mal.

  8. Aufgabe

    Schreiben Sie ein Programm, das die Bücher einer Buchhandlung verwalten kann. Dazu wird eine Struktur benötigt, die die Komponenten Inventarnummer, Titel, Preis, Mwst und Stückzahl für bis zu 100 Bücher aufnehmen kann. Es soll zuerst die Mehrwertsteuer eingelesen werden und dann in einer Schleife die restlichen Bücher. Dadurch braucht die Mehrwertsteuer nur einmal eingegeben werden.

  9. Aufgabe

    Schreiben Sie eine Funktion, die ermittelt, der wievielte Tag im Jahr ein gegebenes Datum ist. Sie soll als Parameter Tag, Monat und Jahr bekommen. Achtung: Schaltjahr!

    Hinweis: Ermitteln Sie die aktuellen Daten für Zeit und Datum mit DosGetDateTime() (OS/2) bzw. GetLocalTime() (Win32). Sehen Sie auch eine Möglichkeit vor, das Datum manuell anzugeben.

    Win32: 

        #include <windows.h>
     ...
    SYSTEMTIME SystemTime;       // Strukturvariable anlegen
     ...
    GetLocalTime(&SystemTime);   // Funktionsaufruf, der die
                                 // Strukturvariable mit dem aktuellen
                                 // Datum und Zeit fuellt

    Die Elemente in der Struktur SystemTime lauten:

    • wYear
    • wMonth
    • wDayOfWeek
    • wDay
    • wHour
    • wMinute
    • wSecond
    • wMilliseconds

  10. Aufgabe

    Schreiben Sie ein Programm, das bis zu 100 Girokonten verwalten kann. Die Daten eines Kontos sollen mind. Name und Anschrift des Inhabers, dessen Geburtsdatum, die Kontonummer und den momentanen Kontostand umfassen.

    Beschränken Sie das Programm zunächst auf die Anlage neuer Konten (Eingabe) und die Ausgabe der Daten der angelegten Konten. Die Anlage neuer Konten umfaßt die Entgegennahme der Daten per Tastatur, das Speichern der Daten in den entsprechenden Variablen und eine Ausgabe der eingegebenen Daten zur Überprüfung auf Tippfehler. Im Falle eines Tippfehlers muß dieser selbstverständlich korrigiert werden können. Die Ausgabe der Daten umfaßt entweder die Ausgabe aller angelegten Kontendaten oder eines beliebigen Kontos (Auswahl per Kontonummer). Die Art der Ausgabe (alle oder ein bestimmtes) muß der Benutzer auswählen können.

    Hinweis: Man kann sich das Leben leichter machen, indem man die Aufgabe zunächst schrittweise aufgliedert und nicht alles auf einmal programmiert.
